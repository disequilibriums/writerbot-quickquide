## Starting a sprint:<br><br>

1. To start a sprint, use the `sprint for` command:

![img](/img/1_sprint_start.png)  

2. You can select the length the sprint will run in minutes. If you click the "+4 more text", you can add additional settings:

![img](/img/1b_sprint_start.png)  

![img](/img/1c_sprint_start.png)  

3. If you want to get notified when new sprints are starting, use the `/sprint notify` command and set it to 'Yes':  

![img](/img/1d_sprint_start.png)
